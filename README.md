# makan-makan-raya-scrabble
A nostalgic Raya themed static HTML mini game/landing page built with vanilla HTML, CSS, and JavaScript. Open with VS Code Live Server to run locally.

## Preview

Classic campaign style microsite featuring:
- Static HTML structure
- Vanilla CSS styling
- Simple JavaScript interactions
- Traditional Raya themed visuals

## Tech Stack

- HTML
- CSS
- JavaScript

## Run Locally

Since this is a static HTML project, it is recommended to run it using VS Code Live Server.

### Using Live Server

1. Install the **Live Server** extension in VS Code
2. Open the project folder
3. Right click `index.html`
4. Select **Open with Live Server**

The project will run on localhost automatically.

Example:

```bash
http://127.0.0.1:5500/
